
# CAFA-6 Protein Function Prediction

This repository contains a machine learning pipeline for the CAFA-6 Protein Function Prediction challenge. The project predicts Gene Ontology terms for protein sequences using LightGBM classifiers, amino acid frequency features, and Gene Ontology hierarchy-based score propagation.
## Project Overview

Protein function prediction is a multi-label classification problem where a single protein can be associated with multiple Gene Ontology terms. In this project, protein sequences are processed from FASTA files and converted into numerical features based on amino acid frequencies. These features are then used to train LightGBM binary classifiers for selected GO terms.

The project also includes a Gene Ontology post-processing step. The GO hierarchy is loaded from the `go-basic.obo` file, and a directed acyclic graph is created using `NetworkX`. Prediction scores are then propagated from child GO terms to their parent GO terms using `is_a` and `part_of` relationships. This helps generate a final submission that follows the Gene Ontology structure.

## Notebooks Included

### 1. LightGBM Prediction Pipeline

This notebook performs the main machine learning workflow:

- Loads protein sequences from FASTA files
- Reads GO term annotations from `train_terms.tsv`
- Removes Cellular Component labels to reduce complexity
- Converts each protein sequence into 20 amino acid frequency features
- Encodes GO terms using multi-label binarization
- Selects top frequent GO terms
- Trains LightGBM binary classifiers
- Predicts GO term probabilities for test proteins
- Generates an initial `submission.tsv` file

### 2. GO Hierarchy Score Propagation

This notebook performs post-processing on prediction scores:

- Loads Gene Ontology data from `go-basic.obo`
- Builds a GO DAG using `NetworkX`
- Extracts `is_a` and `part_of` relationships
- Reads predicted protein-function scores
- Propagates child GO term scores to parent GO terms
- Filters predictions using a score cutoff
- Generates the final `submission.tsv` file
